## 本地评测

### 本地评测数据编写

#### judge.yaml

> 用于指定测试数据集合
>
> ```shell
> tests:[testcase1.yaml,testcase2.yaml,....]
> ```

#### testcase.yaml

> 测试数据

```shell
name: testcase2 # 注意这里需要空空格
data:
  -
    - |
      quit
      quit1
      quit my name
      QUIT
    - haoye!
```

测试指令

```shell
patpat.exe stu -judge 实验序号-20373914-吴文韬 
```
