> 试用 C 语言编写 “孔加工方法链选择”的工艺决策程序

假设有以下几种孔加工方法：

1. 手动钻孔
2. 机械钻孔
3. 激光钻孔

```c
#include <stdio.h>

void select_drilling_method(double diameter, double depth) {
    if (diameter < 0.5) {
        printf("选择手动钻孔\n");
    } else if (depth < 5.0) {
        printf("选择机械钻孔\n");
    } else {
        printf("选择激光钻孔\n");
    }
}

int main() {
    double diameter = 1.0;
    double depth = 10.0;
    select_drilling_method(diameter, depth);
    return 0;
}
```

> 试用 C 语言编写 “装夹方案选择”的工艺决策程序

假设有以下几种装夹方案：

1. 手动装夹
2. 机械装夹
3. 自动装夹

可以根据工件的重量和形状来选择合适的装夹方案

```c
#include <stdio.h>
#include <string.h>

void select_clamping_method(double weight, char* shape) {
    if (weight < 5.0) {
        printf("选择手动装夹\n");
    } else if (strcmp(shape, "简单") == 0) {
        printf("选择机械装夹\n");
    } else {
        printf("选择自动装夹\n");
    }
}

int main() {
    double weight = 10.0;
    char* shape = "复杂";
    select_clamping_method(weight, shape);
    return 0;
}
```
