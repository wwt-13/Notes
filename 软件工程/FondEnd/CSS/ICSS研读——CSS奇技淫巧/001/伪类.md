# 伪类

> 伪类是选择器的一种，它用于选择处于特定状态的元素，比如说被鼠标指针悬停的元素，或是被点击的元素，伪类的特征就是开头为冒号的关键字`:hover`

**常见伪类**

| 伪类           | 元素状态                                 |
| -------------- | ---------------------------------------- |
| `:hover`       | 鼠标悬浮到的元素                         |
| `:active`      | 鼠标点击的元素                           |
| `:focus`       | 获得焦点的元素                           |
| `:first-child` | 第一个子元素                             |
| `:last-child`  | 最后一个子元素                           |
| `:nth-child()` | 第 n 个子元素                            |
| `:invalid`     | 未通过验证的<form>,<input>或其他表单元素 |
