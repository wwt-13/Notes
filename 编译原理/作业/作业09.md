# 作业 09

> 已知以下非分程序结构语言的程序段，画出当斌阿姨改程序段时将生成的有序符号表。

```c
BLOCK:
    REAL X,Y,Z1,Z2,Z3;
    INTEGER I,J,K,LASTI;
    STRING LIST-OF-NAMES;
    LOGICAL ENTRY-ON EXIT-OFF;
    ARRAY REAL VAL(20);
    ARRAY INTEGER MIN-VAL-IDN(20)
    .....
    END OF BLOCK;
```

| 变量名        | 类型    | 维数 |
| ------------- | ------- | ---- |
| ENTRY-ON      | logical | 0    |
| EXIT-OFF      | logical | 0    |
| I             | integer | 0    |
| J             | integer | 0    |
| K             | integer | 0    |
| LASTI         | integer | 0    |
| LIST-OF-NAMES | string  | 0    |
| VAL           | array   | 1    |
| MIN-VAL-IDN   | array   | 1    |
| X             | real    | 0    |
| Y             | real    | 0    |
| Z1            | real    | 0    |
| Z2            | real    | 0    |
| Z3            | real    | 0    |
